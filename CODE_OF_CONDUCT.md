# Code of Conduct

SpeakQuery is a professional engineering project.

Participation is expected to be respectful, technically honest, and
focused on improving the software.

## Expected behavior

- Treat others with professional respect.
- Argue ideas on technical merit, not authority or ego.
- Be clear, direct, and constructive.
- Credit prior work and preserve attribution.

## Unacceptable behavior

- Harassment or intimidation
- Bad-faith participation
- Deliberate misrepresentation of code or intent
- Attempts to erase authorship or extract value through deception

## Enforcement

Project maintainers may remove contributions or participants that
undermine the integrity or sustainability of the project.

This is not a social platform.
It is shared infrastructure.
